# Exa Comp

## Project Specification

- **Framework**: [React](https://reactjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **CSS Framework**: [Tailwind CSS](https://tailwindcss.com/)

## Run Project

1. Instal

   ```bash
   npm i

   ```

2. Run

   ```bash
   npm run dev
   ```
